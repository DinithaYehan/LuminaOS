# LuminaOS

A lightweight web-based desktop environment prototype inspired by modern operating system design. LuminaOS provides a polished boot experience, interactive login flow, and a draggable window system with a built-in terminal app.

## 🌟 Overview

LuminaOS demonstrates a browser-native WebOS-like interface using only HTML, CSS, and JavaScript. The current implementation includes:

- Animated startup/boot screen
- Login workflow that reveals a desktop workspace
- Taskbar with live clock and status indicators
- Desktop icon launching a terminal-style application window
- Draggable window manager with minimize/maximize/close controls
- Terminal command handling for a small set of built-in commands

## 📁 Repository Structure

- `index.html` — Main application shell and interface markup
- `style.css` — Visual design, glassmorphism effects, animations, and layout
- `script.js` — Application logic, boot sequence, interface state, window manager, and terminal behavior

## 🚀 Getting Started

1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Wait for the boot sequence to complete.
4. Click `Enter Workspace` to reveal the desktop interface.
5. Click the `Terminal` icon to open the terminal app.

## 💡 Features

- Smooth animated background and boot loader
- Glass-style login and desktop appearance
- Basic application window system with drag support
- Terminal input with `help`, `whoami`, `skills`, `projects`, and `clear`
- Simple responsive desktop layout for experimentation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
